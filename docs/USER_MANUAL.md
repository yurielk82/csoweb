## 1. 회원가입

### 1.1 가입 절차

1. 로그인 화면 하단의 **"회원가입"** 링크를 클릭합니다.
2. 아래 정보를 순서대로 입력합니다.

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 사업자번호 | O | 10자리 입력 시 국세청에서 자동 인증 |
| 업체명 | O | 회사명 |
| 대표자명 | O | |
| 주소 | O | **"주소 검색"** 버튼 클릭 → 도로명 주소 선택 → 상세 주소 입력 |
| 연락처1 | O | 대표 전화번호 (자동 포맷) |
| 연락처2 | | 추가 연락처 |
| 이메일 | O | 정산서 알림을 받을 이메일 |
| 이메일2 | | 추가 이메일 |
| 비밀번호 | O | 영문 + 숫자 조합, 8자 이상 |
| 비밀번호 확인 | O | 동일하게 재입력 |

3. **"회원가입 신청"** 버튼을 클릭합니다.

### 1.2 국세청 사업자번호 인증

사업자번호 10자리를 입력하면 국세청 공식 데이터로 자동 인증됩니다.

| 인증 결과 | 표시 | 가입 가능 여부 |
|-----------|------|:--------------:|
| 계속사업자 | 녹색 체크 | O |
| 휴업자 | 주황색 경고 | X |
| 폐업자 | 빨간색 경고 | X |

> **계속사업자**로 확인된 경우에만 가입할 수 있습니다.

### 1.3 관리자 승인 대기

- 회원가입 신청 후 **관리자 승인**이 필요합니다.
- 승인이 완료되면 등록된 이메일로 안내 메일이 발송됩니다.
- 승인 전까지는 로그인할 수 없습니다.

## 2. 로그인

### 2.1 로그인 방법

1. **사업자번호**를 입력합니다 (하이픈 자동 입력).
2. **비밀번호**를 입력합니다.
3. **"로그인"** 버튼을 클릭합니다.

### 2.2 최초 로그인 시

관리자가 임시 비밀번호를 발급한 경우, 로그인 후 아래 단계를 거칩니다.

1. **비밀번호 변경** — 새 비밀번호 설정 (영문 + 숫자 8자 이상)
2. **회원 정보 완성** — 업체명, 주소, 연락처, 이메일 입력

두 단계를 모두 완료해야 정산서 조회 화면으로 진입할 수 있습니다.

## 3. 정산서 조회

로그인 후 기본 화면입니다. 월별 정산 내역을 조회하고 엑셀로 다운로드할 수 있습니다.

### 3.1 조회 방법

1. **정산월 선택** — 드롭다운에서 조회할 월을 선택합니다.
2. 데이터가 자동으로 로드됩니다.

### 3.2 검색

- **검색창**에 제품명 또는 거래처명을 입력하고 Enter 또는 검색 버튼을 클릭합니다.
- 부분 일치 검색이 지원됩니다 (예: "아스피" → "아스피린" 검색됨).

### 3.3 표시 항목 설정

- **체크박스**로 화면에 표시할 컬럼을 선택/해제할 수 있습니다.
- 필수 항목(회색 배지)은 해제할 수 없습니다.
- 선택한 항목은 엑셀 다운로드에도 동일하게 적용됩니다.

### 3.4 데이터 구조

정산서는 아래와 같이 계층 구조로 표시됩니다.

```
CSO관리업체 A
  ├─ 거래처 가
  │   ├─ 상세 데이터 행 1
  │   ├─ 상세 데이터 행 2
  │   └─ 거래처 가 합계 (회색 행)
  ├─ 거래처 나
  │   ├─ 상세 데이터 행 3
  │   └─ 거래처 나 합계 (회색 행)
  └─ CSO관리업체 A 총합계 (파란색 행)
```

### 3.5 요약 통계

상단에 3개의 요약 카드가 표시됩니다.

| 카드 | 내용 |
|------|------|
| 총 수량 | 전체 데이터의 수량 합계 |
| 총 금액 | 전체 데이터의 금액 합계 |
| 제약수수료 합계 | **세금계산서 발행 금액** (VAT 포함) |

### 3.6 Notice

관리자가 등록한 안내사항이 주황색 박스로 표시됩니다. 세금계산서 발행일, 마감일 등 중요 공지가 포함되어 있으니 반드시 확인하세요.

### 3.7 엑셀 다운로드

1. 우측 상단의 **"엑셀 다운로드"** 버튼을 클릭합니다.
2. 파일명: `정산서_{정산월}.xlsx`
3. 현재 선택된 표시 항목만 다운로드됩니다.

### 3.8 페이지 이동

- 데이터가 50건을 초과하면 하단에 페이지네이션이 표시됩니다.
- **이전/다음** 버튼으로 이동합니다.

## 4. 월별 수수료 합계

상단 메뉴의 **"월별 합계"** 를 클릭하면 정산월별 수수료 합계를 한눈에 확인할 수 있습니다.

### 4.1 화면 구성

| 영역 | 내용 |
|------|------|
| 요약 카드 | 총 정산월 수, 총 데이터 건수, 수수료 합계 |
| 합계 테이블 | 정산월별 건수와 합계 컬럼 |
| 최하단 행 | 전체 기간 총합계 |

- 가장 최근 정산월은 **파란 배경 + "최신" 배지**로 강조됩니다.
- 수수료/합계 관련 숫자는 파란색으로 표시됩니다.

## 5. 내 정보 관리

우측 상단의 사용자 아이콘 → **"내 정보 수정"** 을 클릭합니다.

### 5.1 조회 가능 정보

| 항목 | 수정 가능 |
|------|:---------:|
| 사업자번호 | X (고정) |
| 회원 유형 | X (고정) |
| 가입일 | X (고정) |
| 업체명, 대표자명 | O |
| 주소 | O |
| 연락처1, 연락처2 | O |
| 이메일, 이메일2 | O |
| 비밀번호 | O |

### 5.2 정보 수정 방법

각 섹션(업체 정보, 주소, 연락처, 이메일)은 **독립적으로 저장**됩니다.

1. 수정할 항목의 값을 변경합니다.
2. 해당 섹션의 **"변경"** 버튼을 클릭합니다.
3. 변경 완료 시 화면 하단에 성공 메시지가 표시됩니다.

> 변경사항이 없으면 버튼이 비활성화됩니다.

### 5.3 비밀번호 변경

1. **현재 비밀번호**를 입력합니다.
2. **새 비밀번호**를 입력합니다 (영문 + 숫자 8자 이상).
3. **새 비밀번호 확인**을 입력합니다.
4. **"비밀번호 변경"** 버튼을 클릭합니다.

## 6. 비밀번호 찾기 / 재설정

### 6.1 비밀번호 찾기

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 링크를 클릭합니다.
2. **사업자번호**와 가입 시 등록한 **이메일**을 입력합니다.
3. **"재설정 링크 발송"** 버튼을 클릭합니다.
4. 등록된 이메일로 비밀번호 재설정 링크가 발송됩니다.

### 6.2 비밀번호 재설정

1. 이메일에서 **"비밀번호 재설정하기"** 버튼을 클릭합니다.
2. 새 비밀번호를 입력하고 확인합니다.
3. **"비밀번호 변경"** 버튼을 클릭합니다.
4. 변경 완료 후 로그인 화면으로 자동 이동합니다.

> **주의사항**
> - 재설정 링크는 **30분** 동안만 유효합니다.
> - 링크는 **1회만** 사용할 수 있습니다.
> - 타인에게 링크를 공유하지 마세요.

## 7. 자주 묻는 질문 (FAQ)

### Q. 회원가입 후 바로 로그인할 수 있나요?

**A.** 아닙니다. 관리자 승인이 완료된 후 로그인할 수 있습니다. 승인 완료 시 등록된 이메일로 안내 메일이 발송됩니다.

### Q. 정산서가 조회되지 않습니다.

**A.** 아래 사항을 확인하세요.

1. **정산월 선택 확인** — 드롭다운에서 정산월이 선택되어 있는지 확인
2. **CSO 매칭 확인** — 사업자번호와 매칭된 정산 데이터가 없으면 조회되지 않습니다. 관리자에게 문의하세요.
3. **해당 월 데이터 없음** — 선택한 정산월에 데이터가 아직 업로드되지 않았을 수 있습니다.

### Q. 엑셀 다운로드 시 일부 컬럼만 포함됩니다.

**A.** 정산서 조회 화면에서 체크박스로 선택한 표시 항목만 엑셀에 포함됩니다. 필요한 항목을 체크한 후 다운로드하세요.

### Q. 비밀번호를 잊었는데 이메일도 기억나지 않습니다.

**A.** 관리자에게 직접 연락하여 비밀번호 초기화를 요청하세요. 초기화된 임시 비밀번호로 로그인 후 새 비밀번호를 설정해야 합니다.

### Q. 사업자번호가 변경되었습니다.

**A.** 사업자번호는 계정의 고유 식별자로 변경할 수 없습니다. 새 사업자번호로 신규 가입 후 관리자에게 기존 계정 삭제를 요청하세요.

### Q. 업체 정보를 수정했는데 정산서에 반영이 안 됩니다.

**A.** 정산서 데이터는 업로드 시점의 정보를 기준으로 합니다. 프로필 수정은 향후 업로드되는 데이터부터 반영됩니다.
